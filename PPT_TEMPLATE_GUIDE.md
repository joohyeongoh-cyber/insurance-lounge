# 보험 전문가 PR 자료 템플릿

## 📄 PPT 템플릿 다운로드

전문가님의 홍보를 위한 맞춤형 PPT 템플릿입니다.

### 포함된 슬라이드:
1. **표지** - 이름, 소속, 연락처
2. **자기소개** - 경력, 전문분야
3. **보험 상품 소개** - 주요 취급 상품
4. **고객 후기** - 실제 고객 리뷰
5. **문의하기** - 연락처, 카카오톡 오픈채팅

### 사용 방법:
1. PPT 파일 다운로드
2. 각 슬라이드의 **[이름]**, **[소속]** 등을 본인 정보로 수정
3. 사진을 본인 사진으로 교체
4. 저장 후 사용!
